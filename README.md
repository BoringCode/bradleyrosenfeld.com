# bradleyrosenfeld.com

> I'm a computer scientist who believes that pixels are people too

This is my personal site. It's hosted as a static site on a VPS. 

## Requirements

- Ruby
- RubyGems
- Bundler
- node.js

## Setup

Install dependencies. 
```bash
bundle
```

Build the site
```bash
jekyll build
```
